# Binary Search Tree Projesi 
[7, 5, 1, 8, 3, 6, 0, 9, 4, 2] dizisinin Binary-Search-Tree aşamalarını yazınız.

Örnek: root x'dir. root'un sağından y bulunur. Solunda z bulunur vb.

```
                4 
             /     \
            /       \
           2         7
          / \       / \
         1   3     6   8
        / \             \
       0   5             9

```
