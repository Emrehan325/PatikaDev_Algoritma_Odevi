# Merge Sort Projesi

* Yukarıdaki dizinin sort türüne göre aşamalarını yazınız.

```

- [16,21,11,8,12,22]

- [16,21]  [11]  [8,12]  [22]

- [16,21] [11]  [8,12]  [22]

```
